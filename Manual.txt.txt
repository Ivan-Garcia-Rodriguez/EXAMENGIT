PROGRAMA ALMACENAMIENTO DE DATOS DE ALUMNOS DE UNA CLASE
========================================================

MANUAL DE INSTRUCCIONES
-----------------------
Este fastuoso programa permite almacenar los nombres de los alumnos de una clase y luego los lista por pantalla. Vaya introduciendo nombres hasta que se le deje de preguntar, luego el programa los listará en el mismo orden enque se metieron. Muy útil, ¿no?...

Relleno de texto para el examen

LISTA DE VERSIONES
------------------
1.0 - El programa permite procesar hasta 10 alumnos
2.0 - Se añade un comentario
3.0 - El programa permite procesar el numero de alumnos que indique el cliente
